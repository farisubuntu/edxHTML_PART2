Labs Overview

This module of the course includes a series of four tutorial style labs and one self-assessment lab. Some lab configuration is required.

The tutorial labs are designed to teach you how to build a foundational layout for your page based on information provided in wireframe designs. These labs will lead you step-by-step through the process of creating your layout, providing you with both written instructions and an explanation of what you are accomplishing.

The final lab in this module is a self-assessment lab. In this lab, you will apply what you've learned during the tutorial labs to implement the design presented in wireframe diagrams for desktop and mobile devices. The self-assessment lab gives you the opportunity to reinforce what you learned without providing the prescriptive step-by-step instructions that are provided in the tutorial labs.

Tutorial Labs 

    1. Planning and Designing a Layout

      It's important to be able to look at the design of a website and start to plan how you would build a layout with CSS. Thinking about the possible issues up front can save you time and cause less issues over the course of the project. In this lab you will review some website designs and use that review to plan your layout.

    2. Build a Semantic HTML and CSS Foundation

      In this lab, you will implement a CSS Reset that defines some base styles and provides a consistent foundation for your web page across browsers. You will also start building our layout based on the wireframe designs that you reviewed in the Planning and Designing a Layout lab.

    3. Analyze Style Guides to Build Your Base CSS

      Now that the foundation for your web site is in place, it's time to build the base CSS. In this lab, you will start by identifying the elements and styles that you should add to the base. After that you will create base styles for body, headings, and other elements.

    4. Use Media Queries to Create a Responsive Grid System

      With your Base CSS in place, it's time to start making your page responsive. In this lab, you will add containers, rows, and columns to your page and then implement a responsive solution that will flip columns when the window displaying your page changes size.
