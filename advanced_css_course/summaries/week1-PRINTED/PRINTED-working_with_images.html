Working with Images

Including effective images in your web page design is crucial and can lead to increased user engagement with your content.

Before you add images to your web page, first prepare the images for the Web using an image editing program. That preparation consists of these three steps:

    1. Resampling the image so that it is at the proper resolution for the Web.
    2. Resizing or cropping the image so that it is the right size on your web page.
    3. Saving the image in one of the three Web image graphics formats: JPEG, PNG, or GIF.

      Tip: For images, JPEG is the best overall option, but the GIF standard allows animated images.

Every image is comprised of pixels. The more pixels per inch the image has, the sharper and more detailed it will appear. Most images on the Web are 72 and 100 pixels per inch.

Tip: Loading images takes time. Since large images can slow down your page, use good judgement regarding the number and size of the images you select for your web page.

HTML Image Tags and Their Descriptions

Use the HTML <img> element to define an image. The <img> tag is empty; it contains attributes only, and does not have a closing tag. Here's the coding:

<img height="##" width="##" src="image.gif">

Use the <img> tag's height and width attributes to define the size of the image. In the coding example above, replace the ## with the corresponding number of pixels for the imageâ€™s height and width. By specifying the actual dimensions of the image, the web page will load with the correct amount of space allocated to the image. Otherwise, the image will download, causing the page to move down and rearrange for every image that subsequently downloads.

To make an image responsive to the container it is in, you will set the width to 100%. The following is a basic CSS example to make your images responsive:

img {
    max-width: 100%;
    height: auto;
}

Use the <img> tag's src attribute to define the URL of the image. Since it is common to store images in a subfolder, include the folder name in the src attribute. If not specified, the browser expects to find the image in the same folder as the web page. Some websites store their images on image servers.

Tip: You can access images from any web address in the world.

Use the <img> tag's alt attribute to define an alternate text for an image if theimage cannot be displayed, due to a slow connection, an error in the src attribute, or if the user is viewing it on a screen reader. A screen reader is a software program that can read what is displayed on a screen. Screen readers are useful to people who are blind, visually impaired, or learning disabled.

Note: The alt attribute is required; a web page will not validate correctly without it.
